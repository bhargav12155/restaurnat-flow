# RealtyFlow 🏡

**AI-Powered Real Estate Marketing & SEO Dashboard**

A comprehensive full-stack web application designed specifically for real estate agents in the Omaha, Nebraska market. RealtyFlow automates content creation, social media management, SEO optimization, and provides advanced video generation capabilities for modern real estate marketing.

## 🛠 **Tech Stack**

### **Frontend**
- **React 18** with TypeScript
- **Vite** for fast development and building
- **shadcn/ui** components on Radix UI primitives
- **Tailwind CSS** with custom design system
- **TanStack Query** for state management
- **Wouter** for client-side routing

### **Backend**
- **Node.js** with Express.js framework
- **TypeScript** with ES modules
- **Drizzle ORM** for type-safe database operations
- **PostgreSQL** with Neon serverless hosting
- **Session Management** with connect-pg-simple

### **External Integrations**
- **OpenAI GPT-5** - AI content generation
- **Facebook Graph API** - Social media posting
- **Instagram Graph API** - Visual content management
- **YouTube Data API v3** - Video upload and management
- **Twitter API v2** - Real-time social engagement
- **HeyGen API** - AI video generation and avatars
- **Google PageSpeed Insights** - SEO performance analysis

## 📦 **Installation & Setup**

### **Prerequisites**
- Node.js 18+ 
- PostgreSQL database
- API keys for integrated services

### **1. Clone Repository**
```bash
git clone https://github.com/bhargav12155/realtyflow.git
cd realtyflow
```

### **2. Install Dependencies**
```bash
npm install
```

### **3. Environment Variables**
Create a `.env` file with the following variables:
```env
DATABASE_URL=your_postgresql_connection_string
OPENAI_API_KEY=your_openai_api_key
FACEBOOK_APP_TOKEN=your_facebook_app_token
FACEBOOK_USER_TOKEN=your_facebook_user_token
INSTAGRAM_ACCESS_TOKEN=your_instagram_access_token
INSTAGRAM_USER_ID=your_instagram_user_id
TWITTER_ACCESS_TOKEN=your_twitter_access_token
TWITTER_ACCESS_TOKEN_SECRET=your_twitter_access_token_secret
TWITTER_CONSUMER_KEY=your_twitter_consumer_key
TWITTER_CONSUMER_SECRET=your_twitter_consumer_secret
YOUTUBE_CLIENT_ID=your_youtube_client_id
YOUTUBE_CLIENT_SECRET=your_youtube_client_secret
HEYGEN_API_KEY=your_heygen_api_key
```

### **4. Database Setup**
```bash
npm run db:push
```

### **5. Start Development Server**
```bash
npm run dev
```

The application will be available at `http://localhost:5000`

## 🎯 **Usage Guide**

### **Content Generation Workflow**
1. **Navigate to AI Content Generator**
2. **Enter property details** (neighborhood, features, price)
3. **Select content type** (social media, blog post, description)
4. **Add relevant keywords** (Omaha-specific terms)
5. **Generate AI content** with one click

### **Social Media Publishing**
1. **Paste or create content** in the social media manager
2. **Select target platforms** (Facebook, Instagram, Twitter, YouTube)
3. **Upload video** (for YouTube integration)
4. **Schedule or post immediately**
5. **Track performance** through analytics dashboard

### **SEO Optimization**
1. **Monitor site health** metrics automatically
2. **Add keyword tracking** for target terms
3. **Review performance** recommendations
4. **Implement suggested** optimizations
