Add Motion and Sound Effects to Your Photo Avatar
Bring your photo avatars to life with dynamic motion and immersive sound effects

Enhance your AI Photo avatar by adding dynamic motion and immersive sound effects! Follow this step-by-step guide to bring your avatar to life.

Add Motion
See detailed API reference
Add motion to your photo avatar by sending a POST request to the API with the avatar's ID.

Request
Response

curl --request POST \
     --url https://api.heygen.com/v2/photo_avatar/add_motion \
     --header 'accept: application/json' \
     --header 'content-type: application/json' \
     --header 'x-api-key: <your-api-key>' \
     --data '{"id": "62d4489300634c3f99c0ec994c31fa76"}'
