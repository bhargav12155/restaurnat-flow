### Components

```

│                    Client Application                        │
│  ┌────────────────┐  ┌────────────────┐  ┌──────────────┐  │
│  │  Page Visit    │→ │ Engagement     │→ │  API Call    │  │

│  └────────────────┘  └────────────────┘  └──────────────┘  │

┌─────────────────────────────────────────────────────────────┐

│  └────────────────┘  └────────────────┘  └──────────────┘  │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                    PostgreSQL Database                       │
│  ┌────────────────┐  ┌────────────────┐  ┌──────────────┐  │
│  │ user_sessions  │  │  property_     │  │ engagement_  │  │
│  │                │  │  interactions  │  │    leads     │  │
│  └────────────────┘  └────────────────┘  └──────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

### Data Flow

1. **User visits page** → Client-side tracker initializes
2. **Session created** → POST `/api/track/session` → Database stores session
3. **User interacts** → Tracker records actions → Database updates
4. **Heartbeat every 30s** → Session time updated
5. **Lead triggers met** → Auto-generate lead → Database stores lead
6. **Admin views dashboard** → GET `/api/analytics/engagement/:agentSlug` → Aggregated data returned

## 🗄️ Database Schema

### 1. `user_sessions` Table

**Purpose**: Track all user sessions with device, location, and behavior data

```sql
CREATE TABLE user_sessions (
  id SERIAL PRIMARY KEY,
  session_id TEXT NOT NULL UNIQUE,          -- UUID v4
  public_user_id INTEGER REFERENCES public_users(id),
  agent_slug TEXT NOT NULL,                 -- Agent this session belongs to
  ip_address TEXT,                          -- IPv4/IPv6 address
  user_agent TEXT,                          -- Browser user agent string
  device_type TEXT,                         -- 'mobile', 'tablet', 'desktop'
  browser_name TEXT,                        -- Browser name
  operating_system TEXT,                    -- Operating system
  country TEXT,                             -- Geo-located country
  city TEXT,                                -- Geo-located city
  first_page_visited TEXT,                  -- Landing page
  last_page_visited TEXT,                   -- Last page visited
  total_time_spent_seconds INTEGER DEFAULT 0,
  total_page_views INTEGER DEFAULT 0,
  total_properties_viewed INTEGER DEFAULT 0,
  total_properties_liked INTEGER DEFAULT 0,
  conversion_type TEXT,                     -- What action converted (if any)
  conversion_value TEXT,                    -- Conversion context
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Indexes for performance
CREATE INDEX idx_sessions_agent ON user_sessions(agent_slug);
CREATE INDEX idx_sessions_session_id ON user_sessions(session_id);
CREATE INDEX idx_sessions_active ON user_sessions(is_active);
CREATE INDEX idx_sessions_created ON user_sessions(created_at DESC);
CREATE INDEX idx_sessions_updated ON user_sessions(updated_at DESC);
```

**Key Fields**:
- `session_id`: Unique identifier (UUID) for each browser session
- `agent_slug`: Multi-tenant support - each agent sees only their data
- `public_user_id`: Links anonymous session to registered user when they log in
- `is_active`: Flag to track if session is currently active

```sql
CREATE INDEX idx_leads_agent ON engagement_leads(agent_slug);
```

**Lead Quality Levels**:

- `warm` (score ≥ 25): Several views, decent time spent

```typescript
app.post("/api/track/session", async (req, res) => {
  try {
    const { sessionId, agentSlug, pageVisited } = req.body;
    
    // Create or update session
    const session = await db.insert(userSessions)
      .values({
        sessionId,
        agentSlug,
        firstPageVisited: pageVisited,
        lastPageVisited: pageVisited,
        ipAddress: req.ip,
        userAgent: req.get('user-agent'),
        totalPageViews: 1
      })
      .onConflictDoUpdate({
        target: userSessions.sessionId,
        set: {
          lastPageVisited: pageVisited,
          totalPageViews: sql`${userSessions.totalPageViews} + 1`,
          updatedAt: new Date()
        }
      })
      .returning();

    res.json({ success: true, sessionId });
  } catch (error) {
    console.error("❌ Error tracking session:", error);
    res.status(500).json({ error: "Failed to track session" });
  }
});
```

```typescript
  try {
    const { propertyId, agentSlug, sessionId, liked } = req.body;

    if (liked) {
      // Add like
      await db.insert(propertyLikes).values({
        propertyId,
        agentSlug,
        sessionId: sessionId || null,
        ipAddress: req.ip || null,
        userAgent: req.get("user-agent") || null,
      });
    } else {
      // Remove like
      await db.delete(propertyLikes).where(
        eq(propertyLikes.propertyId, propertyId) &&
          eq(propertyLikes.sessionId, sessionId)
      );
    }

    res.json({ success: true });
  } catch (error) {
    console.error("❌ Error tracking property like:", error);
    res.status(500).json({ error: "Failed to track like" });
  }
});
```

#### Core Class: `PropertyEngagementService`

```typescript
class PropertyEngagementService {
  private tracker: EngagementTracker | null = null;
  private pageStartTime: number = Date.now();
  private currentPropertyId: string | null = null;
  private heartbeatInterval: NodeJS.Timeout | null = null;
  private apiBaseUrl: string;

  constructor(apiBaseUrl: string = "") {
    this.apiBaseUrl = apiBaseUrl;
    this.initializeSession();
    this.setupPageVisibilityTracking();
    this.startHeartbeat();
  }

  // Public API methods
  public async trackPropertyView(propertyId: string): Promise<void>;
  public async trackPropertyLike(propertyId: string, liked: boolean): Promise<void>;
  public async trackInteraction(type: string, propertyId?: string, value?: string, timeSpentSeconds?: number): Promise<void>;
  public getEngagementSummary(): EngagementSummary;
  public destroy(): void;
}
```

#### Key Features:

1. **Automatic Session ID Generation**
```typescript
private generateSessionId(): string {
  return `session_${Date.now()}_${Math.random().toString(36).substr(2, 9)}`;
}
```

2. **Agent Slug Detection from URL**
```typescript
private getAgentSlugFromUrl(): string {
  const path = window.location.pathname;
  const match = path.match(/\/agent\/([^\/]+)/);
  return match ? match[1] : "unknown-agent";
}
```

3. **Heartbeat System (30-second intervals)**
```typescript
private startHeartbeat(): void {
  this.heartbeatInterval = setInterval(() => {
    if (this.tracker?.isTracking) {
      this.updateSessionTime();
    }
  }, 30000);
}
```

4. **Page Visibility Tracking**
```typescript
private setupPageVisibilityTracking(): void {
  document.addEventListener("visibilitychange", () => {
    if (document.hidden) {
      this.pauseTracking();
    } else {
      this.resumeTracking();
    }
  });

  window.addEventListener("beforeunload", () => {
    this.endPropertyView();
    this.sendPendingData();
  });
}
```

5. **Engagement Score Calculation**
```typescript
public async checkEngagementLevel(): Promise<{
  score: number;
  shouldGenerateLead: boolean;
  reason?: string;
}> {
  let score = 0;
  
  if (sessionTime > 300) score += 20;      // > 5 minutes
  if (viewCount > 3) score += 15;          // Viewed multiple properties
  if (likeCount > 0) score += likeCount * 10;
  if (interactionCount > 5) score += 10;
  if (sessionTime > 600) score += 15;      // > 10 minutes
  
  const shouldGenerateLead = score >= 25;
  return { score, shouldGenerateLead, reason };
}
```

## 🎯 Lead Generation Logic

### Engagement Score Calculation

**File**: `/client/src/lib/engagement-tracker.ts` line ~280

```typescript
public async checkEngagementLevel(): Promise<{
  score: number;
  shouldGenerateLead: boolean;
  reason?: string;
}> {
  if (!this.tracker) {
    return { score: 0, shouldGenerateLead: false };
  }

  let score = 0;
  let reason = "";

  const sessionTime = Math.floor((Date.now() - this.tracker.startTime) / 1000);
  const likeCount = this.tracker.interactions.filter((i) => i.type === "like").length;
  const viewCount = this.tracker.propertyViews.size;
  const interactionCount = this.tracker.interactions.length;

  // Scoring algorithm
  if (sessionTime > 300) score += 20;       // > 5 minutes
  if (viewCount > 3) score += 15;           // Viewed multiple properties
  if (likeCount > 0) score += likeCount * 10; // Liked properties
  if (interactionCount > 5) score += 10;    // High interaction
  if (sessionTime > 600) score += 15;       // > 10 minutes

  // Determine reason
  if (likeCount >= 2) {
    reason = "liked_multiple_properties";
  } else if (sessionTime > 600) {
    reason = "spent_long_time_on_site";
  } else if (viewCount > 5) {
    reason = "viewed_many_properties";
  } else if (interactionCount > 8) {
    reason = "high_interaction_activity";
  }

  const shouldGenerateLead = score >= 25 && reason.length > 0;

  return { score, shouldGenerateLead, reason };
}
```

### Auto-Lead Triggers

Leads are automatically generated when:

1. **Engagement Score ≥ 25** (calculated from above algorithm)
2. **Multiple Properties Liked** (≥ 2 properties)
3. **Long Session** (> 10 minutes)
4. **Many Properties Viewed** (> 5 properties)
5. **High Interaction Count** (> 8 interactions)

### Lead Quality Assignment

```typescript
function determineLeadQuality(score: number): 'hot' | 'warm' | 'cold' {
  if (score >= 40) return 'hot';    // Very engaged user
  if (score >= 25) return 'warm';   // Moderately engaged
  return 'cold';                     // Minimal engagement
}
```

**Quality Definitions**:
- **Hot** (≥40): User has liked multiple properties, spent significant time, likely ready to contact
- **Warm** (≥25): User is actively browsing, showing interest
- **Cold** (<25): User met minimum threshold but engagement is low

### Lead Generation API Call

```typescript
public async generateEngagementLead(): Promise<boolean> {
  if (!this.tracker) return false;

  try {
    const response = await fetch(
      `${this.apiBaseUrl}/api/track/generate-engagement-lead`,
      {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          sessionId: this.tracker.sessionId,
          agentSlug: this.tracker.agentSlug,
        }),
      }
    );

    const result = await response.json();
    return result.success;
  } catch (error) {
    console.warn("Failed to generate engagement lead:", error);
    return false;
  }
}
```

```typescript
function CookieBanner() {

    const consent = localStorage.getItem('tracking-consent');
    if (!consent) setShowBanner(true);

    localStorage.setItem('tracking-consent', 'true');
    setShowBanner(false);
    // Initialize tracking
    initializeEngagementTracking({ page: 'home', agentSlug });
  };

  const declineTracking = () => {
    localStorage.setItem('tracking-opt-out', 'true');
    setShowBanner(false);
  };

  if (!showBanner) return null;

  return (
    <div className="cookie-banner">
      <p>We use cookies to improve your experience.</p>
      <button onClick={acceptTracking}>Accept</button>
      <button onClick={declineTracking}>Decline</button>
    </div>
```

## 📈 Performance Considerations

### Optimization Strategies

1. **Debounced Updates**
   - Heartbeats every 30s, not every second
   - Prevents excessive API calls

2. **Batch Operations**
   - Multiple interactions batched into single API call
   - Reduces network overhead

3. **Async Tracking**
   - Tracking calls don't block UI
   - Uses `async/await` for all API calls

4. **Indexed Queries**
   - All analytics queries use database indexes
   - Fast retrieval even with millions of records

5. **Cached Results**
   - Dashboard caches data for 5 minutes
   - Reduces database load

**Symptoms**: Multiple sessions created for same user

**Cause**: Multiple `initializeEngagementTracking()` calls

**Solution**:
```typescript

  initializeEngagementTracking({ page, agentSlug });
}, []);

  initializeEngagementTracking({ page, agentSlug }); // Duplicate!
}, [page]);

useEffect(() => {
  const tracker = initializeEngagementTracking({ page, agentSlug });
  return () => tracker.destroy();
}, [page, agentSlug]);
```
