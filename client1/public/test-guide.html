2. AI Content Generation Testing

  • Navigate to AI Content Generator (first section in dashboard)
  • Select content type: "Social Media Post"
  • Enter property details: "New listing in Benson neighborhood - 3BR ranch home with updated kitchen and beautiful hardwood floors. Perfect for first-time buyers. $285,000"
  • Add keywords: "Benson real estate", "Omaha homes", "ranch style"
  • Click "Generate Content" button
  • Wait for AI-generated content to appear
  • Verify content includes property details and relevant hashtags
  • Copy generated content for use in social media testing

  • Select the test video: demo-property-tour.mp4 from your files
  • Click "Use This Video" when file is selected
